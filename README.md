### Installation

```bash

```

> âš ï¸ **Beta notice:** Rouxinol.ai is currently in active development. Expect breaking changes and incomplete features. Windows support is available now; macOS and Linux are in progress.

### Quick Start

```bash
# Launch the voice interface
rouxinol start

# Run with a specific language
rouxinol start --lang pt-BR

# Run in offline/on-device mode

```

Once running, just speak naturally:

- *"Open Chrome and go to github.com"*
- *"Create a folder called Projects on the Desktop"*
- *"Take a screenshot and save it to Downloads"*
- *"Set volume to 50 percent"*
